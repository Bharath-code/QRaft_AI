# Financial Model & Pricing Strategy

## 1. Pricing Structure (India / Global)

| Tier | Price (INR/USD) | Features | Psychology |
| :--- | :--- | :--- | :--- |
| **Single** | **₹49 / $4.99** | 1 Image, No Watermark, PNG. | The "Decoy" to make the Pack look good. |
| **Campus Pack** | **₹149 / $14.99** | **5 Images**, Sticker Sheet PDF, Story Bio. | The "Volume Seller" (High Value). |
| **Career Pro** | **₹999 / $29.00** | Lifetime Deal, Recruiter Radar, Deep-Links. | The "Revenue Driver" (High Margin). |

## 2. Unit Economics (Per Sale)

### Scenario: Campus Pack (₹149)
* **Revenue:** ₹149.00
* **AI Cost (5 Gens):** ₹1.25 (Replicate Flux Schnell)
* **Gateway Fee (2%):** ₹3.00
* **Server Allocation:** ₹1.00
* **Net Profit:** **₹143.75**
* **Margin:** **96%**

## 3. Revenue Projections (Milestone: 1,000 Users)
* **Campus Sales (800 students @ ₹149):** ₹1,19,200
* **Career Sales (200 pros @ ₹999):** ₹1,99,800
* **Total Revenue:** **₹3.19 Lakhs**
* **Total Monthly Burn:** ~₹15,000 (Servers/AI)
* **Net Profit:** ~₹3 Lakhs

## 4. Upsell Strategy
1.  **Dynamic Edit:** Charge ₹299 to edit a link *after* printing (for cheap tiers).
2.  **Merch:** Upsell "Print on T-Shirt" via Print-on-demand API later.
